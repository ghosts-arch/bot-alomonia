# bot-lunemesie

Bot discord pour Lunemesie

## Install

```
pip install -r requirements.txt
```

1. Dans le fichier `.env.exemple`, remplacer CLIENT_TOKEN par le token de l'instance
2. retirer l'extension `.exemple` du fichier `.env.exemple`
